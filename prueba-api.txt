Al enviar una petición POST a la ruta http://localhost:3000/registro con el siguiente JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

el servidor recibe la información mediante req.body y responde con un objeto JSON que confirma que los datos fueron recibidos correctamente.

Respuesta obtenida:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}
